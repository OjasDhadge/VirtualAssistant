# VirtualAssistant
A lightweight Python voice assistant named Jarvis that listens to your commands and helps perform everyday tasks like playing music, telling time, fetching Wikipedia summaries, and more — all through your microphone using speech recognition


🚀 Features
🎵 Play YouTube Music — Just say "play [song name]"

🕓 Tell the Current Time — Ask "what's the time?"

🌐 Wikipedia Search — Say "who is [person]"

😂 Tells Random Jokes — Just say "tell me a joke"

🤖 Conversational Responses — Recognizes greetings, intro questions, and requests for help

🎤 Speech Recognition via Microphone

🗣️ Text-to-Speech Output using pyttsx3
